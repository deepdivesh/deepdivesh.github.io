About

My name is Hrvoje, and I'm a full-stack Ruby/JavaScript developeur based in Samobor - a pittoresque town in western Croatia.

I'm hyped about anything related to the web and I think programmers possess magic skills hidden from the general public. They are like wizards in those J.K. Rowling's books.

I'm currently making Blenda, an alternative to Flickr and 500px because I love photography and I don't like the options people have for sharing it. We can do better.

I'm a crazy Linux guy that finds great joy in removing Windows from poor Windows-infected machines. If you're using Linux — hat tip to you, stranger from the internet!

Other things that I love include copyleft, piracy, kindness, privacy, minimalism, compassion and British pronunciation of words like humour and colour.

The complete source code for this blog is available on this Github repo.
