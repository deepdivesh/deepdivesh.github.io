Blogging with Ghost

17 Aug 2014 on random

I've been switching blog engines a lot lately. Currently I'm using the latest hotness on the block, Ghost.

Here are some observations after trying it out.

what is great

One of the main selling points of Ghost is definitely a very sexy UI. It comes with a great theme and feels very snappy and clean. I'm currently under impression that blogs need to have that minimalistic feel, have no distractions and be up to the point. This was probably influenced by Svbtle and Medium.

It's open source, of course, and it's 100% JavaScript - Node and Ember form a wonderful combination here. Reading its source code will definitely teach you a lot, so go ahead.

It comes with a snappy Casper theme, which you can see in action on the Ghost's author blog. I've switched from it since I don't like defaults and I love the look of feel of Medium, so I'm using Readium.

what could be better

Since Ghost is a node application, the authors are expecting you to run it on a server capable of running node. As we all know, hosting costs money and I think it's not necessary for blog to be an actual web application. Instead, it could just be a collection of static pages and I've been using Github pages because of that. I've used it with every blog engine I've tried. They are pretty awesome and completely free.

There is no official support for this, though, so I'm currently using a curl hack as a workaround. It does a fine job, as you can see.

The fact that blog posts are saved in a database is another thing that's itching me. I might be biased with this, because of the problem I have with the upper mentioned scenario, but I love using plain old filesystem as a storage for my posts. Markdown is beautiful and completely open, why prefer databases over it? With Markdown on a filesystem, it's really easy to switch blog engines and since I'm now officially a blog-engine hopper, I consider it important.

I'm not going to just rant about it here, this is something I'm planning to change. If that sounds interesting to you, feel free to reach me on Twitter.

Two heads are better than one.

All in all, I really love Ghost - it's the best blogging experience I've had. Go ahead and try it out, you won't be disappointed!

I'm surely sticking with it for some time.
